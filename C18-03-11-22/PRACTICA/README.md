# 1 día de pronósticos diarios
https://developer.accuweather.com/accuweather-forecast-api/apis/get/forecasts/v1/daily/1day/%7BlocationKey%7D

Devuelve datos de pronóstico diarios para una ubicación específica. Las búsquedas de pronóstico requieren una clave de ubicación. Utilice la API de ubicaciones para obtener la clave de ubicación para la ubicación deseada. De forma predeterminada, se devuelve una versión truncada de los datos de pronóstico por hora. El objeto completo se puede obtener pasando "detalles = verdadero" en la cadena de URL.
